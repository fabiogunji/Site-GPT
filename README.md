# Repositório SITE-GPT
SITE GPT - Assistente Virtual com Azure OpenAI e Google Gemini

O SITE GPT é uma aplicação web interativa desenvolvida com foco em inteligência artificial, acessibilidade e experiência do usuário. O projeto utiliza integração com modelos de IA da Microsoft Azure OpenAI e da Google Gemini API, permitindo alternar dinamicamente entre diferentes provedores de inteligência artificial através de um botão toggle moderno na interface.

A plataforma foi desenvolvida utilizando HTML5, CSS3, JavaScript puro e componentes visuais do Bootstrap, seguindo a identidade visual do SENAI com tema moderno, responsivo e suporte a modo claro/escuro.
